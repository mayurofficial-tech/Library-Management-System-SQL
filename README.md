# 📚 Library Management System (SQL Project)

## 📌 Description
This project is a SQL-based Library Management System that manages books, authors, members, and transactions.

It simulates a real-world library system where users can borrow and return books, while the system tracks availability, fines, and borrowing history.

## 🎯 Objective
The objective of this project is to design a relational database and perform advanced SQL operations including data management, analysis, and reporting.

## 🧱 Database Structure
The database includes the following tables:
- Authors
- Books
- Members
- Transactions

All tables are connected using foreign key relationships to maintain data integrity.

## 🚀 Features
- Create and manage library database
- Insert, update, and delete records (CRUD operations)
- Track book availability
- Manage borrow and return transactions
- Calculate fines
- Perform data analysis and reporting

## 🛠️ Technologies Used
- SQL (MySQL)
- Relational Database Design
